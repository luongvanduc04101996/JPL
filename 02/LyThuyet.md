## Loop  
**While**  
```  
while (điều kiện){  
  Lệnh  
}    
```  
**For**  
* for (đk khởi tạo;đk so sánh;thay đổi đk){  
  Lệnh  
}  
  
* Ví dụ  
for( int i = 1 ; i <5 ; i++ ){  
System.out.println("DUC");  
}  

* __Break__: Để thoát khỏi vòng lập  
* __Continue__: Để tiếp tục vòng lập với đk tiếp theo
