## **Kiểu Dữ Liệu**  
**boolean:** **true** hoặc **false**  
**int:** số nguyên  
**double:** số thực  
**String:** văn bản ("nhập văn bản")  
  
## **Biến, Phép Gán, Kết Hợp**  
**Type** Name;  
**Ví dụ**  
```  
**Biến**  
String ten;  
int soNguyen;  
   
**Phép Gán**  
ten = "Duc";  
soNguyen = 1;  
  
**Kết Hợp**  
String ten = "Duc";  
int diem = 9;  

```  
  
## **Toán Tử**  
-Cộng: +  
-Trừ: -  
-Nhân: *  
-Chia: /  
-Chia lấy dư: %  
-Gán: =  
  
## **Phương Thức**  
**Khai báo:**  
```  
public static void NAME(){  
	STATEMENTS  
}  
```  
**Cách gọi**  
```  
NAME();  
```  
## **If-Else**  
**Cú pháp**
